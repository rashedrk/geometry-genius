Document Object Model

1. What is DOM? what the purpose of DOM

DOM means Document Object Model. it is used to interact with the web. It represents html elements as node. DOM helps to interact with the html elements or web page through javascript. we can change (add,remove) any element into web using dom. or do any action for an interaction with the website.

2. How will you select HTML elements using DOM? Name the DOM methods.

we can use anyof this 5 methods to select any element of HTML using DOM.

  • querySelector()
  • querySelectorAll()
  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()

3. What is event bubble?

when we interact with a webpage an event occurs.like- click,mouse enter,mouse out, hover etc.
when an event occurs the DOM try to catch the event. so it reaches to the node where the event occurs. after capturing the event it propagate the event to its parent elements until it reaches the root element. this technique is called event bubbling.
For example:
if we click an li inside ul then we can see the ul is also clicked. and the container section also clicked. so the event is propagating.

4. What is a callback function and why will you use it?

When we pass a function as an argument of another function then it is callback function. it is used to handle asynchronous operation and events. to handle user interaction with the webpage callback function can be use.
